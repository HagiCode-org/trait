# Trait

Trait is the frontend workspace for `trait.hagicode.com`.

It is designed as a searchable Trait Builder: users can scan indexed behavior signals, assemble a compatible trait stack, and export a self-build blueprint for prompts, bios, or character direction.

## Current scope

- Vite + React + TypeScript frontend skeleton
- Searchable trait atlas with category filters
- Local draft persistence via `localStorage`
- Copy-ready blueprint generation from selected traits
- Responsive single-page builder layout

## Development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run test
npm run build
```
